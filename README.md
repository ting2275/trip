# Shanghai GitHub Pages Site

This repository is prepared for GitHub Pages publishing.

## Files

- `index.html`: main page
- `.nojekyll`: tells GitHub Pages to serve the site as plain static files

## Publish

1. Push this repository to GitHub.
2. In the repository settings, open `Pages`.
3. Set the publishing source to the `main` branch and the `/ (root)` folder.

After GitHub Pages finishes publishing, the site URL will be:

- `https://<your-github-username>.github.io/<repository-name>/`

If the repository name is `<your-github-username>.github.io`, the site URL will be:

- `https://<your-github-username>.github.io/`
